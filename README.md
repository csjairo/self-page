# Personal Page - Jairo Holanda
Uma página minimalista e funcional que reflete meus valores sobre tecnologia, trabalho e tradição.

## Tecnologias
- HTML5 (Semântico)
- CSS3 (Variáveis e Flexbox)
- JavaScript Vanilla (Manipulação de DOM e LocalStorage)

## Funcionalidades
- **Dark Mode:** Persistência de tema através do LocalStorage.
- **Responsividade:** Design adaptável para dispositivos móveis.
- **Tipografia:** Foco em legibilidade e hierarquia visual.
